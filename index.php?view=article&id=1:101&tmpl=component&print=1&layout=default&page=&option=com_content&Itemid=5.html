101年度工作計畫

 臺北市101年度自由軟體桌面應用推動及發展工作計畫

壹、宗旨

一、為宣導智慧財產權，減少非法軟體之使用，於校園中推廣自由軟體，促進校園軟體之多元發展，進而發展本市軟體應用特色。

二、提昇本市中小學教師自由軟體素養，培養自由軟體應用人才，增進教師應用自由軟體建置題庫、進行測驗等支援教學的能力。

貳、    實施對象

本市高中職、國民中小學等學校教職員與學生。

參、    辦理單位

一、           指導單位：教育部

二、           主辦單位：臺北市政府教育局

三、           承辦單位：臺北市大理高級中學、臺北市大直高級中學、臺北市大同高   

級中學、臺北市育成高級中學、臺北市百齡高級中學、臺北市陽明高級中學、臺北市南港高級中學、臺北市景美國民中學、臺北市民權國民中學、臺北市中山國民中學、臺北市重慶國民中學、臺北市國語實驗國民小學、臺北市南湖國民小學、臺北市濱江國民小學、臺北市三民國民小學、臺北市新湖國民小學。

肆、    計畫期程：101月1月1日至101年12月31日止。

伍、    實施活動

一、臺北市101年度自由軟體桌面應用推動與發展小組

小組成員每月召開一至二次會議研商相關工作之推動與發展 (活動一)

　　二、臺北市101年度推動國中小學生學習Scratch程式設計實施計畫

實施本市國中組、國小組scratch師資培育，並辦理scratch作品競賽及建置分享網站、學習社群，以充實教師教學教材與培養學生邏輯推理能力。 (詳見附件活動二)

    三、臺北市101年度中小學校園自由軟體發展研討會

為深植自由軟體在校園的發展基礎，辦理研討會以增進學校各領域教職員對自由軟體應用與發展的了解，進而於校園中倡導自由軟體的應用與推廣。(詳見附件活動三)

四、臺北市101年度推動校園應用Drupal快速建置網站服務實施計畫

Drupal為近來風行的開放源碼社交出版平台，它支援了部落格、評論、投票及建立使用者檔案等功能。運用Drupal，就可以快速地建立具有互動功能的動態網站。(詳見附件活動四)

五、    臺北市101年度高中職及國中小教師製作自由軟體教學教材徵件競賽

為了解本市高中職及國中小教師運用自由軟體進行教學之概況，辦理教學教材徵件競賽，以鼓勵教師進行自由軟體之教學活動。 (詳見附件活動五)

六、臺北市101年度高中職及國中小應用自由軟體海報設計競賽

為拓展自由軟體在校園中的應用，辦理競賽以增進高中職、國中小學生運用自由軟體進行資料搜尋、整合及設計的動機與能力。(詳見附件活動六)

七、    臺北市101年度以開放原始碼為應用核心的open space教學創意計畫

建置以原始碼為應用核心的教學環境，讓教師都能藉由其豐富的教學資源，發揮教學創意，俾增進學生學習效能。(詳見附件活動七)

八、    臺北市101年度ez-go應用知能與實作研習

    培訓本市國中小學教師運用自由軟體的知能，進一步協助學生善用自由學習增進學習之成效。(詳見附件活動八)

九、臺北市101年度校園服務雲端虛擬化建置計畫。

    走向雲端臺北，必先建置主機虛擬。本計畫達成之後，將可節省學校系
　　管人員系統重新建置與管理時間，建立各校私有雲，可利整合學校資源
　　及校際共享！(詳見附件活動九)

十、臺北市101年度自由軟體桌面應用推動與發展小組專業成長實施計畫

　　實施小組成長研習，以增進小組伙伴應用技能。製作自由軟體文宣，以
    利進行推廣。藉由自由軟體社群參訪互通資訊、人力，以達站在巨人肩
    膀，資源共享、減少數位落差的目的。(詳見附件活動十)
