自由軟體教學教材製作競賽

臺北市101年度自由軟體教學教材製作競賽實施計畫

壹、目的

一、增進資訊教育多元化，擴展學生學習視野，鼓勵教師專業成長。

二、徵集自由軟體教學教材，提供教師觀摩，進而拓展自由軟體於教學上的運用。

貳、辦理單位

一、主辦單位：臺北市政府教育局

二、承辦單位：臺北市大直高中、臺北市新湖國小

參、參加對象

競賽分三組，參加對象分別如下：

一、高中職組：本市公私立高中職教師(含代理代課教師)。

二、國中組：本市公私立國中教師(含代理代課教師)。

三、小學組：本市公私立小學教師(含代理代課教師)。

肆、競賽規則

一、競賽內容

    教材內容以介紹某一種或數種自由軟體之操作教學為主，若是介紹共享

軟體，請特別注意版權限制，須付費註冊之各種商業軟體不計分。作業

平台不限，以個人應用之桌面平台應用軟體為主。作品不得有作者姓名、服務單位等個人資訊。

二、競賽期程

1.報名日期：101年4月1至6月30日23時59分。

2.送件日期：於報名時間截止前，自行上傳完成的部份為競賽內容。

    三、參賽人數：每件作品至多三位教師報名。

伍、錄取名額及獎勵：

一、依評審成績，各組錄取特優1件、優選3件、佳作5件。各組並得依參加件數及作品水準酌予調整各類錄取名額或從缺。

二、獎勵方式如下：

特優教案每件發給新臺幣3,000元禮券，優選教案每件新臺幣

2,000元禮券，佳作教案稿酬每件新臺幣1,000元禮券，並發函至學

校建請敘獎與頒發獎狀：特優每位作者小功乙支(第二作者嘉獎二

次，第三作者嘉獎一次)、優選每位作者嘉獎二次(第二作者嘉獎一

次)、佳作每位作者嘉獎一次。獎狀由教育局印製後，送交得獎人學校，由學校轉發給得獎者。

陸、教材評選流程：

一、聘請學者專家與相關的教育人員組成評審團，進行審查，評定作品。

二、審查作品評選標準如下：

1、內容正確　　　　　　　

2、切合主題教學目標，具完整內容、深度與廣度

3、教材製作具創新、實用性、範圍合宜

4、教材符合實際教或學需要　　　

5、訊息設計：文字.圖形.影音及動畫等設計運用合宜

6)介面設計：具親合力.易懂.容易操作，指引清楚.求助訊息

(7)清楚明白、錯誤訊息適時呈現、畫面物件的配置合宜

(8)素材媒體的選擇適當

(9)素材品質優良　　　　　　　　　　　

(10)具多樣性

柒、活動注意事項

一、參賽作品不得以任何形式發包委託他人、廠商等協助建置(或無償委託他人代為設計製作)，相關的軟硬體需取得合法的授權及合理使用，遵守著作權相關之規定，若有違反情事，立即取消參賽資格。若該作品已得獎，將追回相關獎勵。若有違反著作權情事，教師個人自行負責。

二、凡接受學校、教育局或教育部任何型式的補助(金額不拘)、參與公開比賽獲獎者，請勿將同一作品參與本次比賽。

三、獲獎作品教師之著作財產權仍屬教師作者所有，本活動之主辦單位及承辦單位擁有非專屬無償使用權，得以連結方式做教育推廣之權利；亦即凡參賽者必須填寫簽名「報名授權」並依說明於期限交至承辦學校。

四、承辦單位不收取親送案件。所有參賽稿件請自行妥處及定時備份。

捌、觀摩比賽及頒獎：

(一)成績公告：101年7月30日

(二)頒獎方式：獎狀將以公文函轉各校，請各校校長公開轉發、表揚。

(三)成果觀摩：將優良教材連結於本市自由軟體專屬網站供教育同仁觀摩。

玖、經費來源

由教育局相關經費項下支應。

拾、獎勵

本計畫相關承辦人員於計畫執行後，由教育局從優辦理敘獎。

拾壹、本計畫陳  臺北市政府教育局核後實施，修正時亦同。
